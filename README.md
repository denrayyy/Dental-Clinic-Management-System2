# Dental Clinic Management System (DCMS)

This repository contains the source code for the Dental Clinic Management System (DCMS), which includes a web-based admin panel and a mobile application for staff.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [npm](https://www.npmjs.com/)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)

### Installation & Running

#### Admin Web System

1.  **Navigate to the admin project directory:**
    ```bash
    cd dcms-admin
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

    The application will be available at `http://localhost:5173`.

#### Staff Mobile App

1.  **Navigate to the staff project directory:**
    ```bash
    cd dcms-staff
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the Expo development server:**
    ```bash
    npm start
    ```

    This will open the Expo developer tools in your browser. You can then run the app on an Android or iOS simulator, or on a physical device using the Expo Go app.
